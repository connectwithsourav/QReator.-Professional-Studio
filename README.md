# QReator Pro

A professional, highly customizable QR code generator built with React. QReator Pro offers deep visual customization and high-resolution vector/raster exports.

## ✨ Features

* **Deep Customization:** Modify colors, gradients, dot shapes, and corner styles.
* **Logo Integration:** Easily embed your brand logo into the center of the QR code.
* **High-Resolution Exports:** Download your QR codes in multiple formats including PNG, JPG, SVG, and WEBP.
* **Modern UI:** A sleek, responsive interface built with Tailwind CSS and Lucide icons.

## 🛠 Tech Stack

* **Frontend:** React 19, TypeScript, Vite
* **Styling:** Tailwind CSS
* **QR Generation:** `qr-code-styling`
* **Icons:** `lucide-react`

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd <your-repo-directory>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.
